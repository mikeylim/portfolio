# mikedohyunlim.com

Personal portfolio website built with vanilla JavaScript and Tailwind CSS.

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Formspree (contact form)

## Setup

Open `index.html` in a browser — no build step required.

## Deployment

Hosted on GitHub Pages with a custom domain via Namecheap DNS.
